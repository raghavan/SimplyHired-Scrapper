Position:  Software Developer / Data Scientist
Location:  Washington, DC
Job Id:  179-12HMSEC012
# of Openings:   1
The Buller Group has multiple openings for senior developers and database engineers who are interested in data analysis, optimization and presentation.

The ideal candidate loves organizing data and writing applications that analyze and show off results from all kinds of systems including terabyte-scale relational databases and high-volume text and content streams. In this position, the creativity in applying data techniques and tools to problems is more highly valued than specific technology skills.

Personnel who perform these activities are pioneering solutions that work with all kinds of leading-edge commercial and open source database technologies. We work with NoSQL engines like MongoDB and HBase, to the latest relational databases technologies from Oracle (Exadata), IBM DB2 (pureScale) and EMC (Greenplum). Results processing is often performed with Hadoop and possibly Hadapt. The ability to take results from these systems and write applications using Solr, Blacklight, Java Spring, Flex and Ruby is also highly valued.

Combining the skills of a Data Scientist, Data Engineer and Application Designer / Developer, this position provides the opportunity to excel in the high-volume data world.

If you consider yourself a Java/Oracle developer or a Ruby/MongoDB developer and you want to take the next step in your career beyond producing more than just simple web applications, this position is for you.
